Responsive Specifications

Desktop Layout

  • The section has a defined max-width and is typically part of a larger page layout.
  • Form fields may be arranged in a two-column grid for space efficiency where logical.

Tablet Adaptations

  • Maintains the desktop layout but ensures all touch targets are at least 48x48dp.
  • May switch to a single-column layout for form fields if space is constrained.

Mobile Adaptations

  • All form fields stack vertically into a single column.
  • The section footer with action buttons may become sticky at the bottom of the viewport for better accessibility on long forms.
