Responsive Specifications

Desktop Layout

  • The module is presented as a card with a max-width, centered within the main content area of the dashboard.
  • The drag-and-drop zone is large and clearly delineated.
  • The summary report table is designed for horizontal screen real estate.

Tablet Adaptations

  • The layout remains largely the same as desktop, with slightly reduced margins to optimize space.

Mobile Adaptations

  • The card module expands to near full-width.
  • The summary report table becomes horizontally scrollable if its content exceeds the viewport width to prevent breaking the layout.
