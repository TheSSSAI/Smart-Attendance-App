Responsive Specifications

Desktop Layout

  • Filter controls are arranged horizontally in a single row.
  • Uses flexbox with `flex-wrap: wrap` to allow wrapping on narrower desktop screens.
  • Action buttons ('Apply', 'Reset') are aligned to the right.

Tablet Adaptations

  • Maintains the horizontal layout, but filter controls may wrap to a second line if space is limited.
  • Ensures all touch targets are at least 48x48dp.

Mobile Adaptations

  • All filter controls stack vertically, with each control taking up the full width.
  • Action buttons stack vertically at the end of the filter list.

Accessibility Features

Keyboard Navigation

Logical tab order proceeds through each filter control from left to right, then to the action buttons. Dropdowns and date pickers must be fully keyboard-operable.
