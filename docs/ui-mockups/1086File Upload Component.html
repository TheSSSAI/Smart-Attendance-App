Component Variations

Variant Name : "drag_and_drop"

The primary, full-featured variant that provides a large, clearly marked area for users to drag and drop files directly from their desktop. Also includes a clickable link to open the native file browser.

Variant Name : "button_select"

A simplified variant that presents only a button to the user. Clicking the button opens the native file browser. This is useful in contexts where a large drop zone is not practical or necessary.
