Description

The main structural component for the authenticated application experience. It consists of a persistent sidebar for navigation, a top bar for contextual information and actions, and a main content area. The layout is responsive, collapsing the sidebar into a drawer on smaller screens.

Design System Alignment

This organism orchestrates the primary layout using the design system's spacing scale, color palette for backgrounds and surfaces, and responsive breakpoints. It provides the container for molecular components like the Sidebar Navigation and Top Bar.

Responsive Specifications

Mobile Adaptations

  • On screens < 768px, the sidebar is hidden off-screen by default.
  • A hamburger menu icon in the top-left corner toggles the sidebar's visibility.
  • When the sidebar is open on mobile, it appears as a drawer overlaying the main content, accompanied by a dimming overlay on the content area.

Tablet Adaptations

  • On screens >= 768px, the sidebar becomes persistent and is always visible on the left.
  • The main content area occupies the remaining space to the right of the sidebar.

Desktop Enhancements

  • The layout supports a collapsed sidebar state, controlled by a button in the top bar.
  • Transitions for the sidebar width and grid layout provide a smooth open/collapse animation.

Accessibility Features

Keyboard Support

The hamburger menu and sidebar collapse buttons are keyboard focusable and can be activated with Enter/Space. Navigation links within the sidebar are also keyboard accessible.

Screen Reader Support

Focus Management

On mobile, when the sidebar is opened, focus should be programmatically moved to the first item within the sidebar. When closed, focus should return to the toggle button.

Aria Attributes

  • aria-label for toggle buttons.
  • aria-expanded to indicate the state of the mobile sidebar.
  • role='navigation' on the primary navigation list.

Color Contrast

All text and background colors are based on the design system's palette, which meets WCAG 2.1 AA contrast requirements.
