Component: Report Filters Bar

A collection of user-configurable controls for filtering report data. The component is responsive, stacking vertically on smaller screens. The specific filters shown change depending on the report context.

Variant: "Attendance Summary"

Provides filters for Date Range, Team, specific Users, and attendance Status. This is the most common filter set.
