Component Variations

Variant Name: main

The standard app bar for a main screen, typically showing a title and primary actions. It can be adapted for platform conventions (e.g., Android's left-aligned title vs. iOS's centered title).

Variant Name: contextual_actions

An app bar used on a detail or edit screen. It includes a leading navigation action (like a 'Back' button) and contextual trailing actions (like 'Save', 'Edit', or 'Delete').

Variant Name: with_tabs

An extended app bar that includes a row of tabs below the main bar. This is used for screens that have secondary, in-page navigation between related views.
