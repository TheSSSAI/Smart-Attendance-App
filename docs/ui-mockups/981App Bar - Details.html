Description

The App Bar provides a consistent location for screen titles, navigation, and contextual actions at the top of the screen. It adapts to platform conventions and device-specific features like notches.

Design System Alignment

Utilizes the design system's color palette for background and text, typography for titles, and spacing system for padding and icon placement.

Responsive Specifications

Mobile Adaptations

  • Primary target for this component.
  • Adapts height and padding using `env(safe-area-inset-top)` to avoid status bars and device notches.
  • Action icon spacing is optimized for 44x44px touch targets.

Tablet Adaptations

  • Maintains mobile-like appearance as it's a mobile-first component.
  • Spacing may slightly increase for better visual balance on a wider screen, but the core structure remains.

Desktop Enhancements

  • Not applicable. This is a mobile-specific component. The web dashboard uses a different header/sidebar pattern.

Accessibility Features

Keyboard Support

Actionable elements (icon buttons, tabs) are focusable and activatable with keyboard (Tab, Enter/Space).
