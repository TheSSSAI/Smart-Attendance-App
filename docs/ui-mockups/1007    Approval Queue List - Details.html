Responsive Specifications

Desktop Layout

  • Not applicable. This is a mobile-first screen design.

Tablet Adaptations

  • The layout will stretch to fill the tablet width, but the list items will maintain a maximum width to preserve readability, centering the content.
  • Touch target sizes are already sufficient for tablet use.

Mobile Adaptations

  • The layout is optimized for a single-column, full-screen vertical scroll.
  • The contextual action bar is anchored to the bottom for easy thumb access.
  • List items are designed to be scannable and information-dense on smaller viewports.

Accessibility Features

Keyboard Navigation

All interactive elements (list items, checkboxes, buttons) are keyboard accessible in a logical tab order.

Screen Reader Support

Focus Management

Visible focus rings are applied to all interactive elements. When the contextual app bar appears, focus can be moved to it.

Aria Attributes

  • aria-label for list items and icon buttons
  • aria-live for the selection counter
  • role='alert' for any error messages

Color Contrast

All text, icons, and status badges meet WCAG 2.1 AA contrast ratios against their backgrounds.
