Organism: Report Filters Bar

Component Variations

Variant Name : attendance_summary

Standard filter set for general attendance reports. Includes Date Range, User, and Team selectors.

Variant Name : exception_report

Adds an 'Exception Type' multi-select dropdown to the standard filter set, allowing admins to narrow down specific kinds of flagged records.

Variant Name : audit_log

A specialized filter set for the audit log. Includes Date Range, 'Actor' (User), and 'Action Type' selectors.
