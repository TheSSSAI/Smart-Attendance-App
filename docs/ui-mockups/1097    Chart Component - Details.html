Responsive Specifications

Desktop Layout

  • The SVG chart scales to fill 100% of its parent container's width.
  • Axes labels and legends are fully visible.
  • Tooltips appear on mouse hover.

Tablet Adaptations

  • Layout remains the same as desktop, scaling fluidly.
  • Touch targets for data points are large enough for easy interaction.

Mobile Adaptations

  • Chart scales to fit screen width with reduced horizontal padding.
  • Axis label density may be reduced to prevent overlap (e.g., showing every other day).
  • Tooltips are triggered by a direct tap on a data point.

Accessibility Features

Keyboard Navigation

Individual data points are not focusable by default, as the chart is a visual representation. The chart as a whole can be focused, and an ARIA label describes its content.

Focus Management

Focus is managed at the screen level; this component does not trap focus.

Aria Attributes

Color Contrast

All text for labels, axes, and tooltips meets WCAG 2.1 AA contrast requirements against their backgrounds. Chart colors are chosen to be distinguishable, but information is not conveyed by color alone (tooltips provide exact data).
