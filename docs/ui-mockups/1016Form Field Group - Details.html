Description

A fundamental molecular component that groups a label, an input field, optional helper text, and a validation message into a single, cohesive unit. It manages the states (default, error, disabled) for all its constituent parts.

Design System Alignment

This component combines several atomic elements (Text Input, Label) and utilizes design system tokens for spacing, typography, and colors to ensure consistency across all forms.

Responsive Specifications

Mobile Adaptations

  • The vertical stacking of label, input, and helper text is inherently mobile-friendly.
  • Component takes up 100% of its container's width.

Tablet Adaptations

  • Maintains the vertical stacking layout, which works well on tablets in both orientations.

Desktop Enhancements

  • The component can be placed within grid layouts for more complex form arrangements (e.g., two-column forms). Its internal structure remains consistent.
