Dashboard

Welcome to your Dashboard

This is the main content area where screen-specific components and data will be displayed. The layout is responsive and adapts based on the screen size.

On larger screens, you'll see a persistent sidebar on the left. On smaller screens (like tablets), the sidebar will be collapsed by default. On mobile devices, the sidebar is replaced by a bottom navigation bar for a more native mobile experience.

Screen Variations

Variant Name: "Default (Sidebar Open)"

This is the standard layout for desktop screens, featuring the full-width sidebar for easy navigation. The main content area adjusts accordingly.

Demonstrated by the main interactive mockup above.

Variant Name: "Sidebar Collapsed"

On smaller desktop or tablet screens, the sidebar collapses to an icon-only view to maximize content space. It can be toggled open using the hamburger menu in the app bar. The main mockup above is interactive and can demonstrate this state.

Click the hamburger icon in the main mockup to see this state.

Variant Name: "Mobile Layout"

On mobile viewports (<= 768px), the sidebar is hidden entirely and replaced by a bottom navigation bar for a native mobile feel. The main mockup above will automatically switch to this layout if you resize your browser window to be narrow.

Resize your browser window to a mobile width to see this layout.
