Description

A composite form control that combines a text input for filtering with a dropdown list of options. It supports single and multi-selection, asynchronous loading states, and error handling. This component is crucial for selecting items from a potentially large list, such as assigning a supervisor or filtering a report by user.

Design System Alignment

This component utilizes the design system's foundational tokens for colors (primary, neutrals, semantic), typography (body text), spacing, borders (radius, width), and shadows. It is composed of atomic elements like Text Input and incorporates feedback patterns like loading spinners.

Responsive Specifications

Mobile Adaptations

  • The dropdown menu will attempt to render below the input, but JavaScript logic (not demonstrable in static HTML) would be required to flip its position if it would render off-screen.
  • The component takes the full width of its parent container, making it responsive by default.
  • Touch targets within the dropdown options are sufficiently large due to padding.

Tablet Adaptations

  • Behaves identically to the mobile version, adapting to the container width.

Desktop Enhancements

  • The component functions as designed, with hover states providing additional feedback.

Accessibility Features

Keyboard Support

User can Tab to the input. Up/Down arrows navigate options. Enter selects an option. Escape closes the menu.

Screen Reader Support

Uses `role='combobox'` on the input and `role='listbox'` on the menu. Options use `role='option'`. `aria-expanded` indicates the menu state. `aria-activedescendant` tracks the focused option.

Focus Management

Focus remains on the input while the dropdown is open. A visual 'active' state tracks the keyboard-navigated option.

Aria Attributes

  • role='combobox'
  • aria-haspopup='listbox'
  • aria-expanded
  • aria-controls
  • aria-labelledby
  • aria-activedescendant

Color Contrast

All text and UI elements meet WCAG 2.1 AA contrast ratios. Selected items are indicated by font weight and a background color change.
