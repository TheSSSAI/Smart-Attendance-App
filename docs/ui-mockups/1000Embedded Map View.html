Organism: Embedded Map View

This component provides a visual representation of GPS coordinates on a simulated map, complete with loading and error states.

States

Loading State

Loading Map...

Error State

Shown when map tiles cannot be loaded due to a network issue.

Map could not be loaded.
Please check your connection.

Component Variations

Variant Name: "single_pin"

This variant is used when only one location (e.g., a check-in) has been recorded. The map is centered on this single pin.

Check-In 09:01 AM

Variant Name: "dual_pin"

This variant shows both check-in and check-out locations. The map view would typically be centered and zoomed to encompass both pins.

Check-In 09:01 AM
Check-Out 05:03 PM
