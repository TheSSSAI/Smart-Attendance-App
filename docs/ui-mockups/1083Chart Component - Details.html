Description

A responsive, SVG-based data visualization component for displaying trends and summaries. It supports bar and line chart variants, and includes loading and empty states. Accessibility is supported via a visually hidden data table.

Design System Alignment

This component utilizes the design system's color palette (primary, neutrals), typography for labels and tooltips, and spacing tokens for internal layout. It consumes atomic Tooltip and Loading Indicator components.

Responsive Specifications

Mobile Adaptations

  • Reduced padding on the main container.
  • Slightly smaller font size for axis labels to prevent clutter.

Tablet Adaptations

  • Maintains desktop sizing and proportions.

Desktop Enhancements

  • Full interactive tooltips on hover and focus.
  • Spacious layout with clear labeling.

Accessibility Features

Keyboard Support

Interactive elements (bars, points) are made focusable by wrapping them in `` tags with `tabindex="0"`. Tooltips are shown on focus.

Screen Reader Support

Each chart SVG has `role="graphics-document"` and an `aria-label`. A visually hidden `

` with the chart's data is provided as a fallback, linked via `aria-describedby`.

Focus Management

Focus indicators are handled by applying the hover styles to `:focus-visible` pseudo-class.

Aria Attributes

  • role="graphics-document"
  • aria-label
  • aria-describedby

Color Contrast

All text and graphical elements meet WCAG 2.1 AA contrast requirements.
