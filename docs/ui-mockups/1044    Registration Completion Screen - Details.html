Responsive Specifications

Desktop Layout

  • The form is contained within a card with a max-width (e.g., 450px), centered vertically and horizontally on the page.

Tablet Adaptations

  • The layout remains the same as desktop, with the centered card.

Mobile Adaptations

  • The card becomes full-width with reduced horizontal padding.
  • The form may take up the full screen height on smaller devices.
