Responsive Specifications

Desktop Layout

  • N/A - This is a mobile-first screen.

Tablet Adaptations

  • The calendar grid may increase in size, but the layout remains a single-column view optimized for portrait orientation.

Mobile Adaptations

  • Single-column layout with the calendar at the top and the event list for the selected day below.
  • Full-width interactive elements with minimum 48dp touch targets.
  • Font sizes and spacing are optimized for readability on smaller screens.

Accessibility Features

Keyboard Navigation

Logical tab order from header controls (prev, next, today) through the calendar grid days, and then to the event list.

Screen Reader Support

Calendar days are announced with their full date and whether they have events (e.g., 'January 15th, 2025, 2 events'). Selected day's events are read out as a list. All buttons have ARIA labels.

Focus Management

A clear, visible focus indicator (outline) is applied to all interactive elements, including calendar days. When a day is selected, focus moves to the list of events for that day.

Aria Attributes

  • aria-label for icon buttons
  • aria-current='date' for the current day
  • role='gridcell' for calendar days

Color Contrast

All text, event indicators, and interactive elements adhere to WCAG 2.1 AA contrast ratios against their backgrounds.
