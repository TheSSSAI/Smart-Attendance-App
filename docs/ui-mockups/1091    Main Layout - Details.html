Responsive Specifications

Desktop Layout

  • Full-width sidebar is visible by default.
  • Main content area occupies the remaining space.
  • Optimized for screen widths > 1024px.

Tablet Adaptations

  • Sidebar collapses to an icon-only view by default on screen widths between 769px and 1024px.
  • A hamburger menu icon allows toggling the sidebar open and closed.

Mobile Adaptations

  • Sidebar is completely hidden on screen widths <= 768px.
  • A bottom navigation bar appears for primary navigation.
  • Main content area takes up the full screen height between the top app bar and bottom navigation.

Accessibility Features

Keyboard Navigation

Logical tab order from app bar controls to sidebar/bottom navigation, then to main content. Sidebar toggle and all links are keyboard-operable.
