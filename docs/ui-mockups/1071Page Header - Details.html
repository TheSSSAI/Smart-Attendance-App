Description

A layout component that provides a title and optional subtitle for a page or section, along with a container for primary actions like 'Create New' or 'Invite User'.

Design System Alignment

This component uses the design system's typography scale for headings (h3, h4), body text, and spacing tokens for layout. It also consumes the atomic Button component for actions.

Responsive Specifications

Mobile Adaptations

  • On screens < 768px, the header stacks vertically.
  • The title group aligns to the start (left).
  • The actions group becomes full-width, with action buttons stacking vertically.
  • The main title font size is reduced for better visual balance on small screens.

Tablet Adaptations

  • On screens > 768px, the header is a single row with the title on the left and actions on the right.

Desktop Enhancements

  • Maintains the single-row layout with clear separation between title and actions.

Accessibility Features

Keyboard Support

Action buttons are keyboard focusable and activatable.
