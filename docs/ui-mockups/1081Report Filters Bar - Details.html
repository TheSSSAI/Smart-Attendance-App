Description

An organism component that provides a set of controls for filtering large datasets in reports. It is composed of molecular components like Date Range Pickers and Searchable Dropdowns, and adapts its layout for different screen sizes.

Design System Alignment

This component is a composition of molecular components and uses the design system's spacing tokens for layout. Its responsive behavior is a core pattern of the system.

Responsive Specifications

Mobile Adaptations

  • The filters bar stacks into a single column.
  • Each filter group takes the full width of the container.
  • Action buttons also stack vertically at the bottom.

Tablet Adaptations

  • The filters bar becomes a horizontal, wrapping flex container.
  • Filter groups are arranged horizontally.
  • Action buttons appear at the end of the filter row.

Desktop Enhancements

  • Maintains the horizontal layout with more space between elements.
  • Action buttons are aligned to the far right of the container for a clean layout.
