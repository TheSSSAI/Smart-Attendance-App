Component Variations

Variant Name: bar_chart

Use for comparing discrete categories or showing changes over a few distinct time periods. Each bar is interactive, revealing a tooltip with its exact value on hover or focus.

Variant Name: line_chart

Use for visualizing continuous data over time to show trends. Each data point on the line is interactive, revealing a tooltip on hover or focus. An area fill provides additional visual weight to the trend.
