Component: Page Header

Provides a consistent heading structure for main application pages, including a title, optional subtitle, and a container for primary page-level actions.

Component Variations

Variant Name : "Simple"

Use this variant for pages that have a clear title but no primary actions, such as a settings or profile page.

Settings

Manage your personal profile and notification preferences.

Variant Name : "With Actions"

The standard variant for most management pages, providing a title and clear calls-to-action.

User Management

Invite, deactivate, and manage all users within your organization.
