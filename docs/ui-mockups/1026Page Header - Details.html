Description

A layout component that displays the main title of a page or section, an optional subtitle, and a container for primary actions like buttons. It is a key structural element for providing context to the user.

Design System Alignment

Utilizes design system tokens for typography (font sizes, weights), spacing, and colors to ensure consistency with the overall visual foundation.

Responsive Specifications

Mobile Adaptations

  • On screens < 768px, the header stacks vertically, with the title area on top and the actions area below.
  • Action buttons may grow to fill the available width on smaller screens to provide larger touch targets.
  • The title font size is reduced to better fit smaller viewports.

Tablet Adaptations

  • Maintains the desktop side-by-side layout but with reduced horizontal space.
  • Action buttons will wrap to a new line if space is insufficient.

Desktop Enhancements

  • Displays a spacious, side-by-side layout with the title on the left and actions on the right.
