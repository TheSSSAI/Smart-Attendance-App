Description

A responsive, SVG-based data visualization component for displaying trends and comparisons. It supports bar and line chart variants, interactive tooltips on hover/focus, and handles loading and empty data states gracefully.

Design System Alignment

This component heavily utilizes the design system's color palette for data series, text, and backgrounds. It also uses typography tokens for axis labels and tooltips, and spacing tokens for internal padding and layout.

Responsive Specifications

Mobile Adaptations

  • The SVG chart scales fluidly to the container width.
  • Tooltips are positioned to avoid being cut off by the screen edges (requires JS).
  • Axis labels may be angled or thinned out if space is limited to prevent overlap.

Tablet Adaptations

  • The chart continues to scale fluidly. More axis labels can be displayed due to increased width.

Desktop Enhancements

  • Hover-based interactions for tooltips are primary.
  • The chart can display more granular data and higher fidelity details due to the larger viewport.

Accessibility Features

Keyboard Support

Each data point (bar or line point) is made focusable using `tabindex='0'`. Users can navigate between data points using the Tab key.

Screen Reader Support

The main SVG has `role='figure'` and an `aria-labelledby` attribute pointing to the chart title.
