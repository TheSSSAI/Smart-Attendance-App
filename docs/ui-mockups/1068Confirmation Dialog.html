Molecular Component: Confirmation Dialog

A modal dialog for confirming user actions.

Standard Confirmation

Used for confirming non-destructive actions, like saving changes.

Confirm Changes

Are you sure you want to save these changes? This will update the user's profile immediately.

Component Variations

Destructive Action

Used for actions that result in data loss, like deletion. The primary action button is styled destructively.

Delete Team?

Are you sure you want to delete the 'Field Operations' team? This action is irreversible and will remove the team for all members.

Confirmation with Input

Used when an action requires mandatory user input, such as a justification for rejecting a record.

Provide Justification

Please provide a reason for rejecting this attendance correction request. This will be visible to the employee.
