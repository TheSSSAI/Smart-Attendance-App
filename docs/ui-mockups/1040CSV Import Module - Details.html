Description

An organism component that handles the entire user and team bulk import process. It includes states for idle (file selection/drop), uploading progress, server-side processing, and a final summary report showing successes and failures.

Design System Alignment

This organism is composed of several atomic and molecular components, including Primary Button, File Upload (custom), Progress Indicator, and a Data Table for the summary report. It heavily relies on the design system's tokens for color, spacing, typography, and feedback states.

Responsive Specifications

Mobile Adaptations

  • The component takes the full width of its parent container.
  • In the summary report view, the action buttons at the bottom stack vertically to provide larger, easier-to-tap targets.

Tablet Adaptations

  • The component maintains its desktop layout, with summary stats and action buttons remaining in a horizontal row.

Desktop Enhancements

  • The component is designed for desktop, with clear hover states on the dropzone and buttons. The summary report table is optimized for readability on wider screens.
