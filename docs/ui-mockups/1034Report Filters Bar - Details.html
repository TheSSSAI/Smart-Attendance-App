Description

An organism component that groups multiple molecular filter controls (Date Range Picker, Searchable Dropdowns) and action buttons (Apply, Clear) into a responsive toolbar for filtering data tables and reports.

Design System Alignment

This component orchestrates molecular components like Date Range Picker and Searchable Dropdown. It uses design system tokens for spacing, layout, and responsive breakpoints to create a consistent filtering experience across the application.

Responsive Specifications

Mobile Adaptations

  • Filter controls stack vertically for easy scrolling and interaction on narrow screens.
  • Each filter item takes the full width of the container.

Tablet Adaptations

  • The layout may remain stacked or switch to a horizontal, wrapped layout depending on the available width.

Desktop Enhancements

  • Filters are arranged horizontally in a single row, wrapping if necessary.
  • Action buttons are aligned to the far right for a clean, organized layout.
  • Each filter control has a minimum width but can grow to fill available space.

Accessibility Features

Keyboard Support

The entire bar is navigable via keyboard. Users can Tab between each filter control and the action buttons.

Screen Reader Support

The component is wrapped in a `div` with `role="toolbar"` and an `aria-label` to provide context. Each filter control has a programmatically associated `

Focus Management

Focus is managed by the individual molecular components (dropdowns, date pickers) within the bar.

Aria Attributes

  • role="toolbar"
  • aria-label="Report Filters"

Color Contrast

All labels and text content meet WCAG 2.1 AA contrast requirements against the background.
